What is HTML?

HTML (Hypertext Markup Language) is how the majority of content on the internet is built. It acts as the building blocks of a webpage, telling a browser how to structure plain text. Behind most websites is a HTML document that tells the browser what information to display and how to display it.

HTML code is made up of:

  • Elements
  • Tags
  • Attributes

Tags (e.g. a paragraph tag <p>) and their attributes (e.g. a style attribute) make up an HTML element, which forms part of the HTML document.

HTML documents must contain:

  • A doctype declaration
  • A html tag
  • A head tag (contains page name, metadata, etc)
  • A body tag (the info we see on a page)

What is CSS?

HTML by itself looks pretty boring. CSS (cascading style sheets) is used to change the appearance of a page.

CSS can determine:

  • Colour
  • Size
  • Font
  • And more

CSS rules consist of a selector (the HTML tag e.g. <p>), a property (e.g. color) and a value (e.g. red)

CSS rules can be applied to a HTML document in three ways:

  • In-line CSS (targeting a specific HTML tag)
  • Internal CSS (in the head section of a HTML document)
  • External CSS (linked in the head section of an HTML document - this is the most common way for websites as it can be easily applied to multiple pages)
